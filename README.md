# 🎮 Stitch PlayStation Gaming Lounge Manager

A full-featured gaming lounge management system built with React, TypeScript, and Tailwind CSS.

## 🚀 Live Preview

Open `index.html` in the root directory to launch the app, or navigate directly to `kinetic-vault-react/dist/index.html`.

## ✨ Features

- 🔐 **Authentication** – Admin & Customer login (Google, Facebook, Email)
- 🎮 **Room Management** – Real-time room status tracking
- 📅 **Booking System** – Calendar-based reservations
- 🛒 **POS System** – Café menu and orders
- 📊 **Reports & Analytics** – Revenue and session tracking
- ⚙️ **Settings** – System configuration
- 🌍 **Multi-language** – Arabic & English support

## 🛠 Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS v4**
- **Zustand** (state management)
- **React Router v7**
- **Firebase** (auth & database)
- **Recharts** (analytics charts)

## 📁 Project Structure

```
/
```
